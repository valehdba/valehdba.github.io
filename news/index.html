PostgreSQL 17 Released

September 26, 2024
PostgreSQL 17 brings significant improvements to overall performance, with notable gains for in-memory and on-disk sorting. New SQL/JSON capabilities, improved vacuum performance, and enhanced logical replication features.

Website Rebuilt with PostgreSQL

March 2025
Our website has been completely rebuilt with a modern design, member registration, blog platform, event system, and admin panel — all powered by PostgreSQL via Supabase. The source code is open source on GitHub.
Community

Community Founded in Baku

2018
The Azerbaijan PostgreSQL User Group (AZERPUG) was established in Baku as the first dedicated PostgreSQL community in Azerbaijan, with the mission of promoting open-source database technology and knowledge sharing.
